# Password Strength Analyzer

## Project Description
Password Strength Analyzer is a web application developed using Python and Flask.

It analyzes the entered password and determines password strength.

The application also provides suggestions to improve password security.

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript

## Features
- Analyze password strength
- Display password score
- Provide security suggestions
- Show/Hide password option
- Colorful user interface

## How to Run

1. Open project folder in VS Code
2. Open terminal
3. Install Flask

pip install flask

4. Run

python app.py

5. Open browser

http://127.0.0.1:5000

## Output
The application checks password strength and displays:
- Weak
- Medium
- Strong
